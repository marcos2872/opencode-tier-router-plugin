# Plano: Reestruturar Skills, Prompts e Agents

## Objetivo

Tornar skills, prompts e agents:
- **Concisos** — menos texto, mais instrução direta
- **Coerentes** — sem conflitos entre arquivos
- **Adequados a LLM** — instruções que modelos seguem, não ignoram

## Princípio Central

```
compose (orq) → route (seleciona) → actor (dispatcha) → subagente (executa)
```

- **compose**: nunca toca código, só orquestra
- **route**: escolhe agente + modelo baseado no tipo de tarefa
- **actor**: ferramenta de dispatch
- **subagente**: executa o trabalho

## Modelo por Tier

| Tier | Modelo | Uso |
|------|--------|-----|
| explore | opencode/big-pickle | Leitura, grep, git, listagem |
| general-medium | llama.cpp/Nex-N2-mini | Código: fix, refactor, testes |
| general-heavy | opencode/big-pickle | Decisão: arquitetura, specs, review |
